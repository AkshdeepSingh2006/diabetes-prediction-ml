# Diabetes Prediction using Machine Learning

## Overview

This project predicts whether a patient is diabetic or not using machine learning classification algorithms.

The project compares the performance of:

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)

The dataset used is the Pima Indians Diabetes Dataset.

# Dataset

Dataset used:

* Pima Indians Diabetes Dataset

Target column:

* `Outcome`

  * 0 → Non-diabetic
  * 1 → Diabetic

* Accuracy

# How to Run This Project

## 1. Clone Repository

```bash
git clone https://github.com/AkshdeepSingh2006/diabetes-prediction-ml.git
```

## 2. Open Project Folder

```bash
cd diabetes-prediction-ml
```

## 3. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

## 4. Run Jupyter Notebook

```bash
jupyter notebook
```

## 5. Open Notebook

Open:
`Diabetes_prediction.ipynb`

Run all cells step-by-step.
